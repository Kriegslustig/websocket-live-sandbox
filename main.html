Help:

Input

The URL you want to connect to.

Enter: Submit

Textarea

The text you want to send to the socket. Everytime you submit the contents of the text box get sent to the opened WebSocket connection. You may also use JavaScript.

Ctrl/Shift + Enter: Submit

JavaScript

Return: The Return value of the script you submit is sent to the socket.

react(pattern, reactor): A function that calls a callback when the pattern matches a message sent by the server. The pattern may be a String, a RegExp or a match function.

reactor: A function that takes an argument data which contains data sent by the server. Inside the function this.send refers to a function that sends messages to the server.

match: A function that takes a message sent by the server as an argument and returns a boolean value.
